# Retail-Sales-Analysis-PowerBI
# Retail Sales Analysis Dashboard (Power BI)

 Project Overview

Developed an interactive Power BI dashboard to analyze retail sales data for an electronics store.

The dashboard helps business stakeholders understand:

- Sales Performance
- Profitability
- Product Performance
- Promotional Effectiveness
- Regional Sales Trends

---
## Tools Used

- Power BI Desktop
- Power Query
- DAX
---
##  Dataset
Retail sales dataset containing:

- Orders
- Products
- Customers
- Promotions
- Cities
- Sales
- Profit
- Discounts

---

##  Business Questions Answered

✔ Top 5 Products by Sales

✔ Bottom 5 Products by Sales

✔ Sales Trend Analysis

✔ Profit vs Net Sales Relationship

✔ Sales Comparison Between Two Time Periods

✔ Profit Comparison Between Two Time Periods

✔ Units Sold Comparison Between Two Time Periods

✔ Average Discount by Promotion

✔ Sales by City

---

##  Power BI Concepts Used

- Star Schema
- Data Modeling
- Fact & Dimension Tables
- Primary Key / Foreign Key Relationships
- Power Query
- DAX Measures
- KPI Cards
- Interactive Slicers
